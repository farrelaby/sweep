How it works

1

Scan

Point dirsweep at your projects folder. It detects language and package manager for every directory.

2

Browse

See a grouped tree of bloated directories — node_modules, target, .next, and more — sorted by size.

3

Sweep

Toggle what you want gone, hit Enter, and reclaim your disk space. Safe by default (trash), permanent if you want it.

Install

📦

npm

npx dirsweep
🦀

cargo

cargo install dirsweep
🐧

Linux / macOS

curl -sSfL https://github.com/farrelaby/dirsweep/raw/main/install.sh | sh
🪟

Windows

irm https://github.com/farrelaby/dirsweep/raw/main/install.ps1 | iex

Supported targets

Directory Language / Framework
node_modules Node.js
target Rust, Maven (Java)
.next Next.js
__pycache__ Python
.venv / venv Python virtual environments
dist / build JS/TS bundlers, Gradle (Java/Kotlin)
vendor Go, PHP (Composer)
Pods iOS (CocoaPods)
.tox Python (tox)
out Kotlin/Java

Keybindings

Space toggle selection
a select all
d deselect all
Enter confirm delete
j / k navigate
o order by
q quit
