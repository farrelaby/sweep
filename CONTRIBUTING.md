# Contributing

Contributions are welcome! Here's how to get started:

1. Fork the repo
2. Create a feature branch (`git checkout -b feature/amazing-thing`)
3. Commit your changes (`git commit -am 'add amazing thing'`)
4. Push the branch (`git push origin feature/amazing-thing`)
5. Open a Pull Request

## Code style

- Run `cargo fmt` before committing
- Run `cargo clippy` and address warnings
- Run `cargo test` to make sure nothing is broken
- Keep UI changes consistent with the existing Ratatui style
