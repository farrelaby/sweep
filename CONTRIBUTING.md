# Contributing

Contributions are welcome! Here's how to get started:

1. Open an issue or discussion to describe what you'd like to work on — this lets us discuss the approach before you start coding.
2. Fork the repo
3. Create a feature branch (`git checkout -b feature/amazing-thing`)
4. Commit your changes (`git commit -am 'add amazing thing'`)
5. Push the branch (`git push origin feature/amazing-thing`)
6. Open a Pull Request (link to the related issue)

## Code style

- Run `cargo fmt` before committing
- Run `cargo clippy` and address warnings
- Run `cargo test` to make sure nothing is broken
- Keep UI changes consistent with the existing Ratatui style
